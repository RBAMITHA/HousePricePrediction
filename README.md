# House Price Prediction

Internship Project — Week 1

## Overview
Built a regression model to predict house prices based on property features like area, bedrooms, bathrooms, and amenities.

## Files
- `analysis.ipynb` — Full Jupyter notebook (data exploration, cleaning, model building, visualization, insights)
- `Housing.csv` — Dataset used
- `summary.docx` — 1-page summary of findings
- `charts/` — Visualizations (price distribution, correlation heatmap, actual vs predicted)

## Models Used
- Linear Regression (R² ≈ 0.65)
- Random Forest Regressor (R² ≈ 0.61)

## Key Insight
Area and bathroom count are the strongest drivers of house price.
